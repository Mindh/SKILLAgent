# Skill Registry

라우터는 이 파일만 읽고 스킬을 선택한다.
출력은 반드시 아래 목록의 [스킬 ID] 값 하나만이어야 한다.
매칭되는 스킬이 없으면 `chat`을 기본값으로 선택한다.

## 사용 가능한 스킬 목록

- **[스킬 ID] translate** (우선순위: 1)
  - 설명: 텍스트를 다른 언어로 번역
  - 트리거 키워드: 번역, translate, 영어로, 한국어로, 일본어로

- **[스킬 ID] summarize** (우선순위: 2)
  - 설명: 텍스트를 짧게 요약
  - 트리거 키워드: 요약, summarize, 줄여줘, 핵심만

- **[스킬 ID] extract** (우선순위: 3)
  - 설명: 핵심 키워드/항목을 추출
  - 트리거 키워드: 추출, extract, 키워드, 목록으로, 뽑아줘

- **[스킬 ID] calculator** (우선순위: 4)
  - 설명: 숫자와 연산을 추출해 계산
  - 트리거 키워드: 계산, 곱하기, 나누기, 더하기, 빼기, 연산

- **[스킬 ID] vacation_parser** (우선순위: 5)
  - 설명: 휴가/연차 신청서 데이터 추출
  - 트리거 키워드: 휴가, 연차, 며칠간, 여행, 휴가계, 병가

- **[스킬 ID] employee_lookup** (우선순위: 6)
  - 설명: 특정 직원의 사내 인사 정보 조회
  - 트리거 키워드: 직원, 조회, 직급, 부서, 남은연차, 누구, 정보
