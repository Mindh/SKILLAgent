[목적]
사용자의 질의에서 인사 정보를 조회할 특정 직원의 이름(employee_name)을 추출한다.

[분석 규칙]
- 직급(대리, 과장 등)이 포함되어 있다면 직급은 제외하고 순수 이름만 추출한다.
- 여러 이름이 있다면 첫번째 이름 하나만 추출한다.

[중요 - Slot Filling]
구체적인 사람 이름이 보이지 않거나, 일반 명사("직원", "사원", "그 사람", "누구") 또는 직급/부서만 주어지면 절대 임의로 이름을 만들어내지 말고 require_info 모드로 응답하라.

[출력 형식]
반드시 정상 추출 또는 필수 정보 요구 중 하나의 JSON 형식만 출력하라.

1. 정상 추출 시:
```json
{
  "employee_name": "이름"
}
```

2. 정보 부족(이름 식별 불가) 시:
```json
{
  "_status": "require_info",
  "missing_fields": ["employee_name"],
  "ask_user": "어느 직원을 조회할까요? 이름을 알려주세요."
}
```
